# Create PDF from all Drawing Files

## Job Description
Converts all drawings to PDF files. The DXF files are added to Vault as DesignVisualization in a pre-configured folder.

## Installation
Download all files (Jobs and Modules) from the current subdirectory of this repository to your powerJobs directory: *C:\ProgramData\coolOrange\powerJobs*
