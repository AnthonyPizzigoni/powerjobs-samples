# Create PDF and ZIP multiple files

## Job Description
Creates PDF files from drawings in a folder that needs to be specified and compresses all PDF files to a ZIP archive. The ZIP file will be added to Vault as a DesignVisualization.

## Installation
Download all files (Jobs and Modules) from the current subdirectory of this repository to your powerJobs directory: *C:\ProgramData\coolOrange\powerJobs*
